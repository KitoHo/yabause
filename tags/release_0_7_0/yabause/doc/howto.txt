Here's some info about how to build the Yabause documentation
(ie. export docbook files to some other format like HTML)

You'll need the following tools:

* An XSLT processor - I personally use xsltproc:
    http://xmlsoft.org/XSLT/

* XSL stylesheets - I use these:
    http://docbook.sourceforge.net/projects/xsl/
